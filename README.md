## 🚀 Featured Project

### ☁️ CloudAura — Production-Grade Microservices Platform on AWS EKS

> Full e-commerce microservices platform — automated from infrastructure to deployment. Built in 16 days.

**What makes it different:**
- 3-tier node isolation (system / infra / app) with taints and tolerations
- Karpenter Spot provisioning for cost-optimized app nodes
- Terraform-generated secrets — never touched by humans, never in Git
- Self-hosted runner for private VPC database initialization
- One shared Helm chart serving 5 microservices
- Full GitOps — `git push` triggers build → tag → deploy automatically
